# 食譜採購比價網

## 網站服務
- 防疫也要省錢，本網站蒐集不同購物網站的價格資訊，讓讓民眾在家能夠低成本購入他們所需的食材，並依照食譜進行料理。

## 功能簡介
- 使用者可於本網站搜尋食譜，搜尋到食譜後會自動跳出該食譜所需之食材，並於各購物網站抓取食材資訊，讓使用者可以一站式比價，買到最划算的食材。
